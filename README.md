# melodica / otoge

Minimal rhythm game prototype using `uing` for UI and `raudio` for audio.

![melodica](assets/gymnopedie.mp4)
![MISS!](assets/screenshot.png)

## Usage

1. Install shards and dependencies.
2. Run the game:

```bash
shards
make build release=1
bin/otoge
bin/melodica
```

Controls:

- Start: Space
- Lanes: D F J K
